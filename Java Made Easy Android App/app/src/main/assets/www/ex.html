Exercise 1.1

Write a program that will print your initials to standard output in letters that are nine lines tall. Each big letter should be made up of a bunch of *'s. For example, if your initials were "DJE", then the output would look something like:

******           *************        **********
**    **                **            **
**     **               **            **
**      **              **            **
**      **              **            ********
**      **       **     **            **
**     **         **    **            **
**    **           **  **             **
*****               ****              **********

Save the class as Initials.java.

Exercise 1.2

Write, compile, and test a class that displays your full name on the screen. Save the class as FullName.java.

For example if your name is Jay Ramon D. Cruz. Output should look like this:
Jay Ramon D. Cruz

Exercise 1.3

Write, compile, and test a class that displays the pattern of table and chairs on the screen:

X                          X
X                          X
X                          X
X       XXXXXXXXXXXX       X
XXXXXX  X          X  XXXXXX
X    X  X          X  X    X
X    X  X          X  X    X
X    X  X          X  X    X

Save the class as TableAndChair.java.
